## 📋 智能上下文聚合系统（创作前强制执行）

### 【强制执行原则】
**重要**：在每次创作、续写、细纲调整前，必须执行完整的上下文加载，确保AI具备完整的背景信息。

### 【智能上下文加载】

#### 第一步：核心设定强制读取
```
必须读取以下设定文件（强制执行）：
- `设定/人物设定.md` → 所有主要角色的性格、背景、关系
- `设定/世界观设定.md` → 世界规则、力量体系、地理环境
- `设定/异能体系设定.md` → 异能规则、限制、进化方式（如果存在）
- `设定/文风与叙事指南.md` → 写作风格、语言特色、叙事要求
- `设定/AI味句式避免清单.md` → 需要避免的句式和表达方式
```

#### 第二步：智能背景信息聚合
基于创作需求，使用以下命令智能聚合相关信息：

**人物状态追踪**：
- `rg -i "[人物名]" 细纲/ | tail -15` → 了解该人物最近的状态变化
- `rg -i "关系|感情" 细纲/ | tail -10` → 了解人物关系发展

**设定一致性检查**：
- `rg -i "[设定关键词]" 设定/` → 确认相关设定规则
- `rg -i "规则|限制" 设定/` → 确认世界观和力量体系约束

**伏笔网络追踪**：
- `rg -i "伏笔|暗示|铺垫" 细纲/ | tail -10` → 识别需要回收的伏笔
- `rg -i "约定|承诺|誓言" 细纲/` → 追踪重要约定和承诺

#### 第三步：当前章节上下文加载
```
如果正在创作/分析特定章节，必须读取：
- `细纲/章节细纲/第[X]章细纲.md` → 当前章节的详细设计
- `细纲/章节细纲/第[X-1]章细纲.md` → 紧接前情（如果存在）
- `细纲/章节细纲/第[X+1]章细纲.md` → 后续铺垫（如果存在）
```

### 【跨模型兼容文档生成】
完成上下文加载后，生成包含所有必要信息的自包含Markdown文档：
- 包含所有相关设定和背景
- 包含人物当前状态和关系
- 包含必要的情节背景和伏笔信息
- 格式标准化，可直接复制到其他AI模型使用

## 智能适配指南

### 【针对不同作品类型的分析重点】

#### 📚 **都市言情类**
- **核心关注**：情感发展、人物互动、代入感
- **毒点重点**：感情线突兀、角色OOC、第三者插足处理不当
- **受众特征**：多为女性读者，重视情感体验和代入感

#### ⚔️ **玄幻修真类**
- **核心关注**：成长体系、战斗描写、世界观
- **毒点重点**：等级体系混乱、战斗逻辑不合理、主角太弱/太强
- **受众特征**：多为男性读者，重视爽点、成长感和世界探索

#### 🤖 **科幻机甲类**
- **核心关注**：设定严谨性、科技逻辑、未来感
- **毒点重点**：科学设定错误、技术发展不合理、未来感不足
- **受众特征**：偏理性读者，重视逻辑性和设定完整性

#### 👻 **悬疑恐怖类**
- **核心关注**：氛围营造、紧张感、逻辑推理
- **毒点重点**：恐怖效果不足、逻辑漏洞、解谜过程不合理
- **受众特征**：寻求刺激和智力挑战的读者

#### 🌸 **纯爱日常类**
- **核心关注**：温馨感、治愈感、细节描写
- **毒点重点**：过于戏剧化、破坏温馨氛围、角色行为不自然
- **受众特征**：寻求放松和治愈的读者

#### 🎮 **轻小说类**：🔥 **新增**
- **核心关注**：主角智商在线、爽点充足、读者代入感、避免憋屈感
- **毒点重点**：
  - 主角过于被动，缺乏操作空间和反击能力
  - 配角过于"无敌"或"无耻万能"，压制主角光环
  - 过度憋屈的被动妥协，读者代入后感到无力
  - 爽点缺失，无法满足读者娱乐需求
- **受众特征**：多为年轻读者，期待主角智商在线、能力出众、有操作性
- **特殊要求**：必须保持主角能动性，避免过度憋屈的情节

## 核心使用原则

### ⚠️ **智能识别优先原则**

1. **拒绝模板化思维**：不预设任何风格或类型，完全基于实际内容分析
2. **受众导向分析**：始终基于目标读者的特征和期待进行评估
3. **动态标准调整**：不同类型作品使用完全不同的评估标准
4. **适配性建议**：所有建议都必须符合作品的类型特征和受众需求
5. **代入感优先检测**：🔥 **新增** - 自动识别可能导致读者憋屈、无力、被动的情节，优先评估读者代入体验

### 🎯 **精准定位要求**

1. **类型特征识别**：准确识别作品的类型、子类型和风格特征
2. **受众群体画像**：精准构建目标读者的特征和需求
3. **情感逻辑分析**：深入理解该类读者的情感接受逻辑
4. **定制化评估**：基于以上分析进行个性化的毒点评估
5. **智能触发检测**：🔥 **新增** - 根据内容自动触发相应的专业检测模块（如轻小说→憋屈感检测，情感类→代入感检测）
